# Markdown-It

Markdown-it is a powerful and flexible Markdown parser that allows you to convert Markdown syntax into HTML. Itâ€™s widely used for rendering Markdown content on websites, blogs, and documentation.

## Markdown-it Plugins

Markdown-it supports community-written plugins. The Plugins can be loaded like this:

```js
const md = markdownit
  .use(plugin1)
  .use(plugin2, opts, ...)
  .use(plugin3);
```
